Official Linkrex.net plugin. Monetize your website easily with short links.

== Description ==

Monetize your WordPress site with Linkrex. The official plugin allow users to earn more money.

Links monetization (displays an banner ad page when visitors click through links on your website).

== Installation ==

This section describes how to install the plugin and get it working.

In order to use this plugin you will need account on https://linkrex.net.

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard.
2. Search for 'Linkrex Monetizer'.
3. Click 'Install Now'.
4. Activate the plugin on the Plugin dashboard.

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard.
2. Navigate to the 'Upload' area.
3. Select `linkrex-monetizer.zip` from your computer.
4. Click 'Install Now'.
5. Activate the plugin in the Plugin dashboard.

== Configuration ==

Setting up the plugin is super simple and it takes just a couple of minutes to start serving monetized links on your website:

1. Sign up for Linkrex (https://linkrex.net/auth/signup).
2. Install the plugin on your WordPress website.
3. Now goto the 'Settings' and select Linkrex Settings. 
4. Now you will see there are 3 options in the Plugin settings.
   - Token Key: (Important)
   - Domain: (Important)
   - Patterns: (Optional)
5. You can get your token key here (https://linkrex.net/member/tools/quick).
6. Now in second box enter your default website address or just put those domains which you do not want to short.
7. Skip the third box (only for advance users).
8. Save configuration.
9. Done !! The plugin is up and running.

Track your earnings in real-time by logging into the Linkrex dashboard.
